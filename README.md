# simulacionWaymo
Simulación tipo Waymo de auto que esquiva obstáculos hasta llegar a la meta
Este proyecto es una versión hyper-simplificada de un auto Waymo, que bajo una trayectoria de línea recta decide seguir o detenerse basándose en lo que tiene al frente.
Basado en los autos reales Waymo que han desarrollado autos con inteligencia avanzada que pueden determinar variadas situaciones posibles en las calles y reaccionar de manera adecuada.

Cómo correr
```bash
git clone https://github.com/DHC102000/simulacionWaymo.git
cd simulacionWaymo
pip install -r requirements.txt
python waymosim.py
